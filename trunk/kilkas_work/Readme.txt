Информация сканируется с сайта http://www.en.imusic.dk








Исходный файл нужно создавать по следующим параметрам:


Формат - CSV (создается в Excel)


Первые столбцы - Название товара (первых столбцов может быть любое количество)


Последний столбец - штрих-код товара


Исходных файлов можно подложить любое количество









Файл config.ini :


Редактируется как обычный текстовый файл.


Обязательно задать как минимум 1 открытый прокси сервер (найти через Google,Yandex)


Файл "proxy test.csv" позволяет проверить работоспособность прокси сервера,позиции собраные в нем имеют инфо и картинки,
и должны сканироваться.


Чем чаще менять прокси тем лучше.В иделе через каждые 2000 отсканированые позиции.


Если программа зависла прервалась или прокси явно не позволяет сканировать,то программу можно остановить, поменять
любые настройки в файле config.ini и ни чего не удаляя, ни кеша, ни уже готовых результатов запусить заново.


Так же не удаляя, ни кеша, ни уже готовых результатов программу можно запускать несколько раз, по одним и тем же исходным файлам.
Каждый раз конечный файл будет дополняться позициями,которые при первом сканировании могли быть по какой-либо причине пропущены.


При перемещении скрипта,не забыть поправить пути в start.bat


Остальные настройки и коментарии в файле config.ini
